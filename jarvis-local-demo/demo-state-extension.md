# Local Demo State Extension

These sections extend the tab shell with repo-local client-side features.

Rules for these surfaces:
- do not imply live bank sync, market feeds, or external services
- keep calculations local and transparent
- label playful utilities as local-only tools

## Command

- title: Command Center
- status: partial
- core capabilities:
  - natural-language mission intake with structured routing preview
  - local deterministic intent-signal matching across six action modes
  - target-aware focus inference from mission keywords
  - truthful handoff suggestion into the real Builder and Checker pipeline
  - confidence labelling with explicit ambiguity disclosure
- blocked capabilities:
  - no autonomous execution on handoff confirmation
  - no AI inference or backend call during routing
  - no multi-agent orchestration or fabricated runtime activity
- recommended UI content:
  - mission input surface
  - routing preview card with agent, mode, target, and rationale
  - example mission pills
  - handoff confirmation with navigation intent
- warning labels if needed:
  - "Routing is local and deterministic only. No execution is triggered from this surface."
